# Rust Setup Script — `metabooy`

This script installs Rust, configures the environment, and verifies the toolchain.

## Quick Install

```bash
curl -sSL https://raw.githubusercontent.com/MetaBooy/rust-install/refs/heads/main/rust.sh | bash
```
